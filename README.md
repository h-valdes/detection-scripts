# detection-scripts

Scripts for starting the Project "detection"

## start_noetic.sh
Start ROS1 noetic

## start_foxy.sh
Start ROS2 foxy

## start_bridge.sh
Start ROS2 foxy
Start ROS1 bridge
Source the files from the bridge_ws workspace

## camera.sh
Call all the launch scripts to start the detection.
Every script is launched in a different gnome-terminal session.
